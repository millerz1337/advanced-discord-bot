Selamlar, bugün sizlere **all-in-one Discord botumu** tanıtacağım.  
Sadece işimi görsün diye yapmıştım, **minimalist bir yapıya sahip**.  

Bot, **Node.js, Discord.js ve TypeScript** dilleri kullanılarak geliştirilmiştir.  
Botun sorunsuz çalışması için, Discord Developer Portal üzerinden **Bot** section’ından:  
- Presence Intent  
- Message Content Intent  
- Server Members Intent  

aktif olmalıdır.

## Work Modüller:

- `/ban`: Ban a user.  
- `/kick`: Kicks the user.  
- `/mute`: Timeout to user.  
- `/giveaway`: Starts a new giveaway.  
- `/help`: Lists the commands of the bot.  
- `/nuke`: Deletes all messages in the channel.  
- `/ticket`: Sets up the support ticket panel in this channel.
